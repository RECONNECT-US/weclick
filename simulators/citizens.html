Welcome

THIS IS A MOCK / TEST TRAINING APP USED BY WECLICK.

It is designed to help learners practise navigating advice-style information and understanding where to click and what to look for. It does not send any real applications or personal information – it is for training only.

This training app is ready to use. Learner login and trainer delete screens have been removed for hub wiring later.

Find Your Local Citizens Advice

What this training module is for

This module helps you practise using a postcode-style search without using your real postcode.

How this module works

  • Use the practice box to type in a pretend postcode format.
  • The app will check if it looks like a UK-style postcode (for example “S1 2AB”).
  • No real locations are looked up – this is only to practise typing and checking carefully.
  • Save the module when the learner is confident using this type of search.

Practice: Typing a postcode-style code

Type a practice postcode such as S1 2AB or LS10 3CD. Do not use your real home postcode.
