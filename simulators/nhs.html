Help & information

This NHS app training simulator is designed by WECLICK to help people practise common digital health tasks in a safe space before using the real NHS app.

  • Learners use their name and a 4-digit code to save progress on this device.
  • Trainers can add or remove learners using the Manage learners area.
  • Each module shows key steps and tips for using that part of the app, alongside a visual mock screen.

This is a training simulation only. It does not connect to the NHS, your GP surgery or any real medical system. No real personal data should be entered.
